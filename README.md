# MultiShop

A microservice-based e-commerce application built with ASP.NET Core, featuring modern architecture patterns and technologies.

## Tech Stack

**Backend:** ASP.NET Core, Ocelot API Gateway, Identity Server, SignalR  
**Databases:** MongoDB, PostgreSQL, MSSQL, SQLite, Redis  
**Architecture:** Microservices, Onion Architecture  
**DevOps:** Docker, Portainer  
**Other:** JWT Authentication, Rapid API, Google Cloud Storage 

## Services

Catalog · Basket · Order · Payment · Discount · Cargo · Comment · Message · Images · SignalR · Identity Server · API Gateway

## Getting Started

```bash
git clone https://github.com/enginbas/MultiShop.git
cd MultiShop
docker-compose up -d
```

## Requirements

- .NET 6.0
- Docker
